# 01 产品需求

## 文档目的

说明华哥自动节点订阅池的产品定位、核心目标和第一阶段边界。

## 当前版本范围

- 软件名称：华哥自动节点订阅池
- 项目定位：华哥自用后台，不是 SaaS，不面向客户或粉丝登录。
- 核心目标：自动采集公开节点、自动检测可用性、自动维护订阅池。
- 自动化主流程：采集公开节点 -> 解析候选节点 -> 检测可用性 -> 更新节点池 -> 刷新订阅输出。
- 手动备用原则：自动流程失败时保留手动触发、手动刷新和手动配置入口。
- 本版本只做骨架，不做真实业务。
