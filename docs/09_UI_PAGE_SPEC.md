# 09 UI 页面规范

## 文档目的

记录后台页面结构、菜单和占位交互要求。

## 当前版本范围

当前后台只允许 7 个菜单：总览、采集管理、检测管理、订阅管理、内核管理、统计数据、系统设置。

v0.3.0 页面展示要求：

- 采集管理：显示 GitHub 线索数量、节点池总数、待检测节点数、最近解析摘要，并提供“解析最近 GitHub 线索”和“手动导入文本解析”入口。
- 检测管理：待检测数量来自节点池状态，继续提示真实检测在 v0.4.0 实现。
- 统计数据：显示真实节点池统计、协议统计、来源统计、地区统计和最近节点列表。
- 节点列表只展示脱敏节点，不展示完整 raw 节点。

v0.3.1 页面小修：

- 统计数据页面提供“清空节点池”按钮。
- 清空节点池前必须二次确认，确认文案说明只清空本地节点池测试数据，不删除配置文件。
- 采集管理页面搜索结果链接列使用固定宽度“打开”按钮，避免竖排显示。

v0.4.0 检测管理页面：

- 显示 Xray 是否安装、Xray 路径、是否正在检测、队列数量、最近检测时间和最近错误。
- 显示待检测、检测中、可用、不可用、暂不支持、错误节点数。
- 提供“检测 1 条待检测节点”“检测 5 条待检测节点”“刷新检测状态”按钮。
- Xray 未安装或检测运行中时，检测按钮不可重复触发。
- 显示最近检测历史：时间、检测数量、可用、不可用、暂不支持、错误。
- 页面提示当前只支持 Xray-core 基础检测，不支持 sing-box / Mihomo，不生成订阅。

v0.4.2 检测管理页面：

- 当存在 `unsupported` 节点时，展示“当前节点参数暂不支持，可等待后续版本增强兼容性”提示。
- 节点列表继续展示 `failureReason`、`responseMs`、`lastTestedAt`，不展示 raw 节点。

v0.4.3 检测管理页面：

- 页面版本显示更新为 `v0.4.3`。
- 检测管理页面展示安全 debug 摘要：协议、传输、security、flow、代理类型、检测核心。
- TLS/Reality 握手类失败需要显示更清楚的提示：Reality 握手失败，可能是配置映射或节点不可达。
- 节点列表继续禁止展示 raw 节点、uuid、publicKey、server 和完整临时配置。

v0.4.5 统计数据页面和节点列表：

- 节点列表显示当前状态、是否手动确认、手动原因、最近检测时间和失败原因。
- 每行提供“标记可用”“标记不可用”“恢复待检测”按钮。
- 三个手动操作按钮点击前均需二次确认，操作成功后刷新统计和节点列表。
- 页面提示“自动检测可能误判，手动确认可用的节点后续也可进入订阅池。”
- 节点列表仍只展示脱敏节点，不展示 raw 节点。

v0.5.0 订阅管理页面：

- 页面只显示一个“安全订阅链接”，不再显示 raw 订阅链接和 base64 订阅链接两个并列入口。
- 显示当前订阅节点数、目标节点数、最低保底节点数、最后生成时间、当前状态和风险提示。
- 操作按钮只保留“生成/刷新安全订阅”和“复制安全订阅链接”。
- v0.7.1 起，二维码区域在订阅可访问时显示二维码，并提供“下载二维码”按钮。
- 页面提示“当前版本只生成一个安全订阅链接，订阅内容由后台缓存输出。”

## 后续待补充内容

补充真实表格、筛选、状态提示、错误展示和保存交互规范。

## v0.6.0 订阅管理页补充

- 订阅管理页仍然只显示一个“安全订阅链接”。
- 页面增加自动刷新状态：已开启/未开启、刷新间隔、上次自动刷新时间、下次自动刷新时间、最近自动刷新结果、warning 和错误信息。
- 操作按钮仍然只保留“生成/刷新安全订阅”和“复制安全订阅链接”。
- 不恢复 raw 订阅链接、base64 订阅链接或二维码复制/下载按钮。
- 页面不展示 raw 节点或完整订阅内容。

## v0.6.1 订阅管理页补充

- 页面不显示 `safeSubscriptionUrl` 文本。
- 页面不显示完整 URL、`/sub/{token}` 或 token。
- 页面将链接状态展示为“安全订阅：已生成 / 未生成”。
- “复制安全订阅链接”按钮仍复制完整订阅地址。
- 复制失败时提示“当前浏览器不支持自动复制，请手动在接口中查看。”

## v0.6.2 订阅复制交互

- 复制按钮优先使用 `navigator.clipboard.writeText`。
- 如果 clipboard API 不可用或失败，回退到 textarea + `document.execCommand("copy")`。
- 复制成功提示“已复制安全订阅链接”。
- 复制失败提示“自动复制失败，请检查浏览器权限或使用 HTTPS 后重试”。
- 未生成订阅链接时提示“请先生成安全订阅链接”。
- 完整订阅地址只用于复制，不渲染到页面。

## v0.6.3 订阅管理页补充

- 页面只显示“公开订阅域名：已配置 / 未配置”。
- 页面不展示完整公开订阅链接。
- 页面不展示 `/sub/{token}` 或 token。
- 复制按钮使用 `publicSubscriptionBaseUrl + safeSubscriptionUrl` 生成剪贴板内容。
- 未配置公开订阅域名时提示“请先配置公开订阅域名 SUBSCRIPTION_PUBLIC_BASE_URL”。

## v0.6.4 订阅管理页补充

- 订阅管理页新增“重置安全订阅链接”按钮。
- 点击前必须确认“重置后旧订阅链接将失效，确定继续吗？”。
- 重置成功后提示“安全订阅链接已重置，请重新复制新链接”。
- 页面仍不显示完整链接、`/sub/{token}` 或 token。
- 页面按钮只允许“生成/刷新安全订阅”、“复制安全订阅链接”、“重置安全订阅链接”。

## v0.7.0 订阅有效期展示

- 订阅管理页显示订阅有效期状态：有效、已过期或未生成。
- 页面显示到期时间、剩余时间、有效期天数和最近续期时间。
- 页面新增“续期订阅有效期”按钮。
- 点击续期前确认“续期后订阅有效期将从现在开始重新计算，确定继续吗？”。
- 续期成功提示“订阅有效期已续期”，失败提示“订阅续期失败，请检查服务状态”。
- 页面仍不显示完整链接、`/sub/{token}`、token、raw 节点或 raw/base64 两个链接。

## v0.7.1 订阅二维码展示

- 订阅管理页在订阅状态卡片下方显示“订阅二维码”区域。
- 二维码可用条件：订阅已生成、公开订阅域名已配置、可复制公开订阅地址已准备好、订阅未过期且公开订阅可访问。
- 二维码内容使用 `publicSubscriptionBaseUrl + safeSubscriptionUrl` 生成，但页面不显示完整链接文本。
- 未生成订阅时提示“请先生成安全订阅。”。
- 未配置公开订阅域名时提示“请先配置公开订阅域名 SUBSCRIPTION_PUBLIC_BASE_URL。”。
- 订阅过期时提示“订阅已过期，请先续期。”。
- 下载二维码按钮下载 `huage-secure-subscription-qr.png`。
- 页面仍不显示完整链接、`/sub/{token}`、token、raw 节点或 raw/base64 两个链接。

## v0.8.0 公开领取页

- `/claim` 页面不显示后台侧边栏、后台标题、后台菜单或节点池基础版徽章。
- 页面标题为“华哥免费节点订阅领取”。
- 页面说明为“请输入视频中公布的领取口令，验证后复制订阅链接。”。
- 输入框占位提示“请输入领取口令”。
- 主按钮为“验证并领取”。
- 口令验证成功后显示“口令验证成功，请复制订阅链接。”，并显示“复制订阅链接”按钮。
- 复制成功提示“已复制订阅链接”。
- 复制失败提示“自动复制失败，请手动检查浏览器权限或使用 HTTPS 访问。”。
- 底部提示“本订阅为免费分享，节点可用性会随时间变化，请以实际连接为准。”。
- 公开页不显示完整链接、`/sub/{token}`、token、后台接口、节点数量、节点来源、检测结果、日志、服务器信息或 raw/base64 内容。

## v0.8.1 领取页复制规则

- `/claim` 页面验证口令时只调用 `POST /api/claim/verify`。
- `/claim` 页面不调用 `/api/subscription/status`。
- 口令验证成功后，将后端返回的 `copyableSubscriptionUrl` 存在前端状态里。
- 点击“复制订阅链接”时复制 `copyableSubscriptionUrl`。
- 页面不显示 `copyableSubscriptionUrl`、完整订阅链接、`/sub/{token}` 或 token。
- 口令错误、订阅未生成、订阅过期、公开域名未配置等错误直接展示后端 `message`。

## v0.8.2 领取页防刷和二维码下载

- `/claim` 页面验证按钮点击后进入 loading 状态，避免重复点击。
- 口令错误时显示后端 `message`。
- 如果后端返回 `remainingAttempts`，页面显示“还可尝试 X 次”。
- 如果后端返回 `CLAIM_TOO_MANY_ATTEMPTS` 和 `retryAfterSeconds`，页面显示冷却提示。
- 口令正确后仍只显示“复制订阅链接”按钮，不显示完整链接或 token。
- 订阅管理页不再直接显示二维码图片或可见 canvas。
- 二维码区域只显示“订阅二维码：已生成 / 不可用”和原因说明。
- 点击“下载二维码”时临时生成并下载 `huage-secure-subscription-qr.png`。
- 订阅管理页不显示完整订阅链接、`/sub/{token}`、token、raw 节点或 raw/base64 链接。
